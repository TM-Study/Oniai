# Oniai

## 概要

- 技術キャッチアップを目的とした既存マッチングアプリの模倣

## ディレクトリ構成

```plain
/Oniai
|-- client : フロントエンドのソースコードを配置。Next.jsで実装。
|-- server : サーバーサイドのソースコードを配置。Spring Bootで実装。
```
